# SpawnerManager

Unity Tools to Manage Spawn point, type and trigger position.


## How to :
    - Import package 
    - Create a Trigger GameObject prefabs with SM_Trigger scripts and collision component with trigger option
    - Give this prefabs to the tools
    - You can use it !
