# SpawnerManager

Unity Tools to Manage Spawn point, type and trigger position.


## How to :
    - Import package 
    - Create a Trigger GameObject prefab with SM_Trigger script and collision component (with trigger checked)
    - Give this prefab to the tools
    - You can use it !
